# Retail Sales Performance Dashboard

## Project Overview
This project presents an interactive Retail Sales Performance Dashboard developed using Power BI. The dashboard provides insights into sales, profit, customer behavior, payment methods, and product performance.

## Dashboard Preview

![Dashboard Screenshot](dashboard.png)

## Key Features

- KPI Cards for:
  - Total Sales
  - Total Profit
  - Total Quantity
  - Average Rating
  - Total Transactions

- Interactive Filters:
  - Date
  - City
  - Branch
  - Gender
  - Product Line
  - Payment Method

- Visualizations:
  - Monthly Sales Trend
  - Sales by City
  - Payment Method Mix
  - Profit by Product Line
  - Average Rating by Product Line
  - Sales by Customer Type

## Tools Used

- Power BI
- Power Query
- DAX

## Key Insights

- Identified top-performing cities based on sales.
- Analyzed customer purchasing behavior.
- Evaluated product line profitability.
- Examined payment method preferences.
